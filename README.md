- ⚡ Next.js 14 (App Router) + TypeScript
- 🎥 Framer Motion animations
- 📱 Fully responsive (mobile-first)
- 👆 Touch gestures + ⌨️ keyboard nav
- 🧠 Skill showcase w/ CountUp FX
- 🧊 Dark glassmorphism UI

- Next.js 14
- TypeScript
- Framer Motion
- TailwindCSS
